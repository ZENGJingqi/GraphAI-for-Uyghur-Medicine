# Data Directory Guide

This folder keeps the necessary stored data for the repository.

## Core Stored Files

- `full_prescription_graphs_with_labels.pt`
  Stored full-corpus graph tensor for all available samples.

- `gat_model.pth`
  Previously trained GAT model weights.

- `UHF_Cluster_Dummies_Unique.csv`
  Original cluster dummy file from the earlier workflow.

- `UHF_Label_Matrix.csv`
  English normalized label matrix for all 480 prescriptions.

## Source Tables

- `UHF_UHP.tsv`
  Full prescription-to-herb relation table.

- `UHF_TCMT.tsv`
  Prescription-to-terminology mapping used in the original label-generation context.

- `UHP_Encoder.tsv`
  Herb-level encoded node features.

- `UHP_Medicinal_properties_encode.tsv`
  Herb-to-property edge table.

- `Uighur_herbal_formulas.tsv`
  Prescription metadata table.

- `Uighur_herbal_pieces.tsv`
  Herbal piece metadata table.

- `Test_input.xlsx`
  Example Excel input for prediction simulation.

## Generated Output Note

Prediction outputs and attention outputs are generated by the workflow when needed.

They are not kept as stored repository result files.
